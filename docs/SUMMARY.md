# Documentation Summary

agent-skills — A CLI tool and skill library that installs standardized AI agent workflow skills and shared behavioral rules into any of 40 supported AI coding agents with a single command.
Monorepo built with TypeScript, Bun, and Turbo; published as `@buiducnhat/agent-skills` on npm.
